# NAME

App::Greple::jq - greple -Mjq module

# SYNOPSIS

    greple -Mjq ...

# DESCRIPTION

This is a module for [App::Greple](https://metacpan.org/pod/App::Greple) command to provide interface for [jq(1)](http://man.he.net/man1/jq) command.
